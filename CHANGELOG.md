## [1.0.0] - 2025-11-01

### Added
- Initial release of Claude Code Task Manager
- Cross-session intelligent task management system
- Multi-language support (English and Traditional Chinese)
- Persistent storage for tasks
- Priority and status management
- Tag system for task categorization
- Import/Export functionality
- CLI interface with comprehensive commands
- Background process management (inspired by `/bashes`)
- Programmatic API for Node.js/TypeScript projects

### Features
- Task CRUD operations (Create, Read, Update, Delete)
- Session tracking across different Claude Code sessions
- Background process management with status monitoring
- Task filtering by status, priority, and tags
- Export/Import tasks for backup and restore
- Localization support (en, zh-TW)
